Welcome to Your Time Planning Tool

Mom Schedule is a time planning tool for busy Mothers. Mom Schedule has a calendar, offers event editing and creating task cards.

As a site user you can: Login, Create Tasks, Categorize, See only (In)Complete, Order by Due date, Overview your Month.

View and Edit an individual task so that you can read the details and adapt it to your needs.

This appication can be useful not only to mothers, fathers, but for any caretaker or just a busy person.

Contact us for any questions or comments about the app.

Login

Register so that you have a unique and private schedule and then login so that you can view your own schedule.

Create Tasks


Create tasks with Title, Category and the due dates. Each category has an image helping to visually identify it.

Categorize


Filter by Category, so you can see the tasks that are relevant to your current area of activity.

See only (In)Complete

Filter and see the tasks that you have already completed or only those to work on. Toggling function allows to revert the complete/incompete status. Can come quite handy if you need some schedule adjustments.

Prioritize by Due date

Order by Due date, so you do not miss important deadlines.

Overview your Month

See your monthly schedule to overview your tasks. Tasks headings are marked on a Calendar.
